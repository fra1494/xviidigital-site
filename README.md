# XVII Digital Site

Sito statico pronto per GitHub Pages.

## Pubblicazione su GitHub Pages
1. Crea una repository, ad esempio `xviidigital-site`.
2. Carica tutti i file contenuti in questa cartella.
3. Vai su **Settings > Pages**.
4. Source: **Deploy from a branch**.
5. Branch: `main`, folder `/root`.
6. Salva e attendi la pubblicazione.

## APK Duo
Quando vuoi rendere disponibile la beta Android:
- inserisci il file APK in `downloads/`
- rinominalo esattamente `duo.apk`
- il pulsante “Android Beta APK” funzionerà automaticamente.
